# Role: 당신은 React, Tailwind CSS, Zustand 기반의 2D 게임 UI 및 페이퍼돌(Paper Doll) 아바타 장착 시스템 구현 전문가입니다.

# Objective
사용자가 올린 메이플스토리 '토벤머리' 캐릭터 이미지(`public/avatar_base.png`)를 기본 아바타로 설정하고, 상점에서 장비 아이템(모자, 무기, 갑옷 등)을 구매하여 인벤토리에서 장착하면 캐릭터 위에 해당 장비가 시각적으로 입혀지는 '아바타 장착 시스템'을 구현하고자 합니다. (몬스터 사냥 시스템은 구현하지 않습니다.)

# Technical & Design Requirements

1. **Zustand 스토어 확장 (`useStore.js`)**
   - 현재 캐릭터 정보 스토어에 장착된 장비 상태(`equippedItems`)와 인벤토리 상태(`inventory`)를 추가하세요.
     * `equippedItems: { HAT: IItem | null, WEAPON: IItem | null, ARMOR: IItem | null }`
   - **액션(Actions) 추가:**
     * `buyItem(item)`: 소지한 골드(GP)를 차감하고 인벤토리에 아이템 추가.
     * `equipItem(itemId)`: 해당 슬롯에 장비를 장착하고, 기존 장비가 있다면 해제(Unequip).
     * `unequipItem(slot)`: 장비를 해제하여 인벤토리로 되돌림.
   - **장비 장착 효과:** 장비 장착 시 현실 퀘스트 완료 시 얻는 경험치(XP)나 스탯 획득량이 일정 비율(예: 1.1배, 1.2배) 증가하는 패시브 버프 로직을 가볍게 연결해 주세요.

2. **페이퍼돌 아바타 컴포넌트 (`AvatarDisplay.jsx`)**
   - 아바타 영역은 상대 좌표(`relative`) 컨테이너로 설정합니다.
   - **Base Layer (최하단):** `public/avatar_base.png` (토벤머리 캐릭터)가 중앙에 깔끔하게 배치됩니다.
   - **Equipment Layers (상단 절대좌표 배치):** 장착된 장비의 이모지(Emoji)를 캐릭터의 정확한 신체 위치 위에 `absolute` 좌표로 배치합니다.
     * 머리 (`HAT`): 캐릭터 머리 위 (예: 👑, 🧢, `top-[12%] left-[46%] text-3xl`)
     * 오른손 (`WEAPON`): 캐릭터의 손 위치 (예: 🗡️, 🪄, `top-[55%] left-[22%] text-4xl`)
     * 상체 (`ARMOR`): 캐릭터 몸통 위치 (예: 👕, 🛡️, `top-[62%] left-[43%] text-3xl`)
   - **애니메이션:** 장착/해제 시 Framer Motion을 활용하여 장비가 "뿅!" 하고 나타나는 부드러운 스케일 업 애니메이션 효과를 주세요.
   - 아바타 주변에 현재 장착된 [머리], [무기], [갑옷] 슬롯을 동그란 칸으로 보여주는 전용 슬롯 UI를 추가하세요.

3. **인벤토리 및 가상 상점 컴포넌트 (`InventoryShop.jsx`)**
   - 대시보드 하단이나 적절한 영역에 탭 메뉴로 구성된 [모험가의 가방(인벤토리)]과 [마을 상점]을 만드세요.
   - **상점 판매 목록 예시 (기본 데이터 세팅):**
     * 모자: "수련생의 두건" (👑, 가격: 100 골드, 효과: 지능 획득량 1.1배)
     * 무기: "모험가의 목검" (🗡️, 가격: 150 골드, 효과: 근력 획득량 1.1배)
     * 갑옷: "수련용 도복" (👕, 가격: 200 골드, 효과: 모든 스탯 획득량 1.05배)
   - 상점에서 골드로 아이템을 사고, 인벤토리에서 '장착' 버튼을 누르면 화면의 토벤머리 아바타에 즉시 이모지가 예쁘게 입혀지도록 구현하세요.

기존의 깔끔한 한국어 UI 대시보드 레이아웃을 해치지 않고 자연스럽게 어우러지도록 전체 코드를 작성하거나 필요한 컴포넌트 코드를 리팩토링해 주세요.
